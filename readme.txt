卡牌diy制作器。

网址：http://www.lsyweb.top:8081/

目前只做了秘境对决，以后可能会做出杀戮尖塔等等游戏的...

前端使用layui框架，后端springboot，不涉及数据库。

本地的E:/cardimg/保存用户上传的卡图(Linux中是/cardimg)
本地的E:/cardimg/cardbase保存各种卡底

==========================
所以只需要建E:/cardimg文件夹，并把resources里面的image文件夹拷到这个文件夹里，重命名cardbase
(绝对路径E:/cardimg/cardbase)就可以运行了
